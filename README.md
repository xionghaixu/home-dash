# home-dash

## 项目简介

`home-dash` 是阶段一已收口的个人与家庭网络存储系统后端项目，基于 Spring Boot 构建，提供文件管理、大文件分片上传/下载、系统信息展示、视频播放等服务端能力。

## 当前定位

- 技术栈：JDK 21 + Spring Boot 3.3.4 + MyBatis
- 数据库：H2 2.2.224（默认）/ MySQL（可选）
- 适用场景：单用户、家庭主人、局域网优先
- 阶段状态：阶段一已完成并收口

## 阶段一已实现能力

- 文件列表查询、排序和面包屑路径导航
- 文件夹新建、重命名、删除、移动、复制
- 文件上传、下载、删除、重命名
- 文件类型分类浏览
- 最近上传列表
- 传输任务列表与上传面板
- 文件夹树选择弹窗
- 系统信息统计
- MD5 分片校验与断点续传支持
- 统一响应结构与全局异常处理

## 规划方向

- 阶段二：预览与检索整理
- 阶段三：媒体中心与 NAS 体验
- 阶段四：数据治理与任务中心
- 阶段五：家庭设备与备份页面
- 阶段六：登录与安全页面

## 技术栈

- Spring Boot：3.3.4
- JDK：21
- 数据访问：MyBatis
- 数据库：H2 2.2.224 / MySQL
- 构建工具：Maven
- 部署方式：Docker / 脚本启动

## 快速开始

```bash
mvn clean package
bin/startup.sh
```
