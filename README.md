# home-dash

## 项目简介

`home-dash` 是 HomeDash 项目的后端服务，基于 Spring Boot 3 + MyBatis-Plus，提供文件管理、分块上传、下载、系统统计等能力。

## 当前状态（阶段一至阶段三）

- 阶段定位：已完成核心网盘、预览检索整理、媒体中心三阶段能力
- 适用场景：单用户、家庭主人、局域网优先的家庭媒体/文件中心
- 当前能力清单：
  - 阶段一：文件列表、目录切换、面包屑、上传/下载、重命名、删除、移动、复制、传输列表、系统信息
  - 阶段二：搜索与筛选、详情增强、图片/文本/音频预览、收藏/标签/备注等轻量整理能力
  - 阶段三：首页媒体聚合、图片/视频/音频三类媒体列表与基础浏览播放体验

## 技术栈

- JDK 21
- Spring Boot 3.3.4
- MyBatis-Plus
- H2 2.2.224（默认）/ MySQL（可选）
- Maven

## 快速开始

```bash
mvn clean package
bin/startup.sh
```
